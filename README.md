# Colorbolt Website

Static landing page, Terms of Service, and Privacy Policy for [colorbolt.net](https://colorbolt.net).

## Structure

| File | Purpose |
|------|---------|
| `index.html` | Landing page — hero, features, store badges |
| `terms.html` | Terms of Service |
| `privacy.html` | Privacy Policy |
| `style.css` | Shared dark-themed styles |
| `public/` | Static assets + Cloudflare Pages config |

## Deployment

Hosted on Cloudflare Pages with direct Git integration. Push to `main` to deploy.
